# Analise_Dados_Socioeconomicos_Municipios
Este trabalho analisa indicadores socioeconômicos de municípios brasileiros a partir de dados fictícios, utilizando IDH, PIB per capita, taxa de desemprego e Índice de Gini. Foi aplicado o z-score para construir um índice de vulnerabilidade, permitindo identificar desigualdades e comparar os municípios.
